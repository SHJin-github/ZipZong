##### 회원가입, 로그인 기능 제시

- 회원가입시 필수 항목에는 id(username), password, email, nickname이 들어감
  - nickname은 중복 허용이 안되고 랜덤하게 생성할 수 있어야 함
- 유효성 검사 처리는 FE, BE 둘 다 진행함
